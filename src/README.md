#### Umlauts

This template supports German umlauts in the main text. If you need umlauts on the title page, for example in the document title, you may have to use the native LaTeX way of writing:

| Umlaut | LaTeX way |
|--------|-----------|
| ä      | {\\"a}    |
| ö      | {\\"o}    |
| ü      | {\\"u}    |
| Ä      | {\\"A}    |
| Ö      | {\\"Ö}    |
| Ü      | {\\"Ü}    |
| ß      | {\ss}     |
